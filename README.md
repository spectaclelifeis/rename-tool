# 📂 Image Rename Tool
파이썬 `tkinter`로 제작된 **사진 파일(이미지) 전용** 이름 일괄 변경 도구입니다.
<br> 
<br>
## 📥 다운로드 및 실행 방법
1. 오른쪽 **Releases** 섹션에서 최신 버전의 파일을 다운로드하세요.
2. `rename.exe` 파일을 내려받아 즉시 사용할 수 있습니다.
<br>

## 📂 파일 이름 변경 규칙 (상세 가이드)

본 프로그램은 선택한 폴더 내의 파일들을 아래 규칙에 따라 **두 자리 숫자(`01`, `02`...)**를 기반으로 일괄 변경합니다.

### 1. 시작 번호 (두 자리 고정)
* 입력한 시작 번호부터 **항상 두 자리 형식**으로 번호가 매겨집니다. (001, 002 등 세 자리 이상은 지원하지 않으며 오직 두 자리로만 고정됩니다.)
  * 예: `1` 입력 시 👉 `01`, `02`, `03` ...
  * 예: `10` 입력 시 👉 `10`, `11`, `12` ...

### 2. 시작 문구 (옵션)
* **문구 입력 시**: `문구_번호.확장자` 형태로 변경됩니다.
  * 예: 문구 `답`, 번호 `1` 👉 **답_01.jpg, 답_02.jpg...**
* **문구 미입력 시**: 숫자만으로 이름이 변경됩니다.
  * 예: 문구 (빈칸), 번호 `1` 👉 **01.jpg, 02.jpg...**

### 3. 변경 예시 표
| 설정 항목 | 입력값 예시 | 변경 후 결과 (예시) |
| :--- | :--- | :--- |
| **숫자만 사용** | 번호: `1` | `01.png`, `02.png` |
| **문구 + 숫자** | 문구: `정답`, 번호: `1` | `정답_01.jpg`, `정답_02.jpg` |
| **중간부터 시작** | 문구: `Test`, 번호: `10` | `Test_10.pdf`, `Test_11.pdf` |
